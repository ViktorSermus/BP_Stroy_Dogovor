# Подразделения: строительный договор (витражное остекление)

Единый справочник ролей для построения BPMN и документации процессов.

| Код | Подразделение | Роль в процессе |
|-----|----------------|-----------------|
| **РП** | Руководитель проекта | Руководит и контролирует все процессы по договору от начала до конца. |
| **ПТО** | Отдел ПТО | Регистрирует договор; далее ведёт работу по закрывающим документам по договору. |
| **Финансы** | Финансовый отдел | Ведёт учёт финансовых потоков в системе **1С7** (приходы, расходы и связанный учёт). |
| **ПО** | Проектный отдел | Разрабатывает проектную и сметную документацию; ведёт исследования и работы по **геодезии** объекта. |
| **ПДО** | Производственно-диспетчерский отдел | Контроль и диспетчеризация: закупка (через задания ОС), сроки, **производственный** и **монтажный** графики; согласование с **РП**. |
| **ОС** | Отдел снабжения | По заданию **ПДО**: закупка материалов, доставка по указанным графикам в указанные склады. |
| **Производство** | Производство | Сборка конструкций по графику **ПДО** при наличии материалов от **ОС** (поставки по графику **ПДО**). |
| **МП** | Монтажное подразделение | Монтаж конструкций на объекте после их получения на объекте, по графику **ПДО**. |

## Связи по цепочке (кратко)

- **РП** — сквозной контроль; ключевые согласования с **ПДО**.
- **ПДО** — задаёт рамки сроков и графиков для **ОС**, **Производства** и **МП**.
- **ОС** исполняет закупку и логистику **только в рамках заданий ПДО** (и согласованных графиков).

## Примечание

Дополнительные роли (например ТЭО, товарный учёт, отдельные контуры 1С) при появлении в процессе заносятся в этот файл отдельными строками.
